# proyecto-tienda-alura
Este proyecto analiza el desempeño de cuatro tiendas para identificar la mejor opción de venta para su dueño. Se evalúan ingresos, distribución de ventas, satisfacción del cliente, productos destacados y costos de envío, integrando métricas financieras y operativas para determinar la tienda que pueda ser vendida sin generar efecto negativo

# Funcionalidades del Proyecto
El proyecto incluye un conjunto de análisis y visualizaciones orientadas a comprender el comportamiento comercial de cada tienda:

1. Análisis Financiero
  - Cálculo del total de ingresos por tienda.
  - Comparación porcentual entre tiendas.
  - Gráfico circular representando el aporte de cada tienda al ingreso total.

2. Análisis de Ventas
  - Identificación del producto más y menos vendido por tienda.
  - Gráfico comparativo (barras) del desempeño de estos productos.
  - Ranking de las tres categorías más vendidas por tienda.

3. Satisfacción del Cliente
  - Cálculo del promedio de calificación por tienda.
  - Gráfico recomendado (línea o burbujas) para comparar satisfacción.
  - Evaluación de la percepción del cliente frente al desempeño comercial.

4. Costos de Envío
  - Promedio de costo de envío por tienda.
  - Comparación visual entre tiendas.
  - Identificación de oportunidades de mejora logística.

5. Recomendación Final
Incluye:
  - Justificación de la tienda recomendada para la venta.
  - Análisis estructural de fortalezas y debilidades por tienda.
  - Recomendaciones para potenciar las 3 tiendas restantes.

# Instrucciones de Uso
1.- Asegúrate de contar con los archivos CSV
- Cada archivo contiene: Producto, Categoría del Producto, Precio, Costo de Envío, Calificación
2.- Ejecución del Análisis
- Importa las librerías necesarias
- Carga los datasets
- Ejecuta cada bloque de análisis (Cada sección incluye código directo, simple y comentado para facilitar el aprendizaje)

# Resultados Esperados
Al finalizar la ejecución del proyecto, obtendrás:
  - Una visión integral del desempeño de cada tienda.
  - Gráficos que facilitan la toma de decisiones.
  - Identificación de la tienda óptima para su venta (Tienda 1).
  - Recomendaciones prácticas para mejorar las otras tres tiendas
