# proyecto-tienda-alura
Este proyecto analiza el desempeño de cuatro tiendas para identificar la mejor opción de venta para su dueño. Se evalúan ingresos, distribución de ventas, satisfacción del cliente, productos destacados y costos de envío, integrando métricas financieras y operativas para determinar la tienda que pueda ser vendida sin generar efecto negativo
